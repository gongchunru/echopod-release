# EchoPod

Public downloads and release notes for EchoPod.

EchoPod is a macOS desktop podcast app focused on listening, transcripts, and AI summaries.

## Download

Installers are published from the Releases page:

https://github.com/gongchunru/echopod-release/releases

## Repository scope

This repository is only for public release notes and downloadable installer files.

The application source code is kept in a private repository. Do not upload source archives, project files, build scripts, credentials, or internal documentation here.

## Maintainer workflow

1. Build and sign the installer from the private source repository.
2. Create a versioned release in this repository, for example `v1.0.0`.
3. Upload only the installer package and any public checksums or release notes.
